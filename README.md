Unlocking Societal Trends in Aadhaar Enrolment and Updates

This repository contains the complete analytical workflow developed for the UIDAI Hackathon.
The project leverages aggregated Aadhaar enrolment and update datasets to uncover societal trends, operational patterns, anomalies, and predictive indicators that can support informed decision-making and system improvements.

Repository Contents

analysis.ipynb
Jupyter Notebook containing end-to-end data preprocessing, exploratory analysis, visualisations, anomaly detection, and predictive modelling.

hackathon.py
Python script version of the complete analysis pipeline to ensure reproducibility and modular execution.

README.md
Project documentation and usage overview.
